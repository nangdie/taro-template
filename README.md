# 3.X Taro-Template [查看模板](https://github.com/nangdie/taro-template)
## 简介
> 什么是Taro ? Taro是能够一次开发多端的框架，如：微信/京东/百度/支付宝/字节跳动/ QQ 小程序/H5 等。

该模板已在多个项目上使用、已将所有依赖升级到目前最新，整个模板全面拥抱了Hooks，将你可能用到的技术都封装到一起。
## 技术栈
> 结构主要是参考了Umi

- Taro
- TypeScript 
- SCSS 
- Redux
- Flyio
